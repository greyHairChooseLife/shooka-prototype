## 채널 특성

### 채널 정체성

머니코믹스는 주식·부동산·재테크 정보를 웹툰·애니메이션 형식으로 시각화해 전달하는 채널.
여러 출연진과 제작진이 참여하는 프로그램형 콘텐츠가 주를 이룸.

### 타겟 시청자

재테크에 관심 있는 2030. 투자 정보를 딱딱하지 않게 습득하고 싶은 입문~중급 투자자.

### 콘텐츠 스타일

- 복잡한 금융 개념을 캐릭터와 스토리로 시각화
- 웹툰 원작 기반의 애니메이션 퀄리티가 차별점
- 정보 밀도가 높고 실용적인 내용 중심
- 분량은 보통 10~20분대로 압축

### 시청자 댓글 문화 (액션 아이템 도출 시 반드시 고려)

이 채널 시청자들은 밈·반어법을 주류 언어로 사용한다.

- "나 망했다 ㅋㅋ", "역시 난 안 돼" 같은 자조적 표현은 공감의 유머이지 부정 반응이 아님
- 댓글의 표면적 의미보다 채널 문화적 맥락을 고려해 해석할 것
- 단, 반어로 포장된 실질적 비판(오류 지적, 주제 불만)은 진지한 피드백으로 처리

### 시청자가 기대하는 것

## 지시사항

이 영상에 달린 **{{totalCommentCount}}개 댓글의 종합 분석 결과**를 바탕으로, 다음 콘텐츠 기획에 실질적으로 도움이 되는 액션 아이템을 **최대 2개**만 도출하라.

### 원칙

- 카테고리 수에 맞추려 하지 말 것. 전체 댓글 데이터를 종합적으로 검토한 뒤 가장 신호가 강하고 실행 가능한 것 2개를 선택한다.
- 비중이 낮은 카테고리(10% 미만)는 단독 근거로 쓰지 않는다.
- 여러 카테고리에서 같은 방향의 신호가 교차 확인될 때 우선적으로 채택한다.
- 각 아이템은 반드시 채널 특성과 기획 방향성에 부합해야 한다.

### rationale 작성 기준

단순한 댓글 인용이 아니라, **{{totalCommentCount}}개 댓글 전체 분포에서 이 신호가 얼마나 강한지**를 수치와 함께 서술해야 한다:

1. 해당 카테고리의 비중(%)과 댓글 수
2. 댓글들에서 반복되는 구체적 패턴 또는 표현
3. 왜 이것이 다음 콘텐츠 기획에서 가장 중요한 신호인지
4. 제안하는 액션이 이 신호에 어떻게 응답하는지

### 액션 아이템 유형

- **다음 영상 주제**: 댓글에서 반복 요청되거나 강한 관심이 확인된 후속 주제
- **스토리텔링·전달 방식**: 이번 영상에서 효과적이었거나 개선이 필요한 구조
- **신뢰도·전문성 관리**: 오류 지적 패턴이 뚜렷할 때의 대응 방안
- **제목·썸네일**: 댓글 반응에서 드러난 기대와 실제 내용의 간극을 해소하는 방향

### 카테고리 설명

- 주제 만족도: 주제·내용에 대한 만족/불만 반응
- 전문성·정정: 전문 지식 보완 및 사실 오류 지적
- 콘텐츠 요청: 다음 주제 직·간접 요청
- 시청자 공감: 자기 상황 대입·감정 표출
