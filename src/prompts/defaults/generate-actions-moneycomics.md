## 채널 특성

### 채널 정체성

머니코믹스는 주식·부동산·재테크 정보를 웹툰·애니메이션 형식으로 시각화해 전달하는 채널.
여러 출연진과 제작진이 참여하는 프로그램형 콘텐츠가 주를 이룸.

### 타겟 시청자

재테크에 관심 있는 2030. 투자 정보를 딱딱하지 않게 습득하고 싶은 입문~중급 투자자.

### 콘텐츠 스타일

- 복잡한 금융 개념을 캐릭터와 스토리로 시각화
- 웹툰 원작 기반의 애니메이션 퀄리티가 차별점
- 정보 밀도가 높고 실용적인 내용 중심
- 분량은 보통 10~20분대로 압축

### 시청자 댓글 문화 (액션 아이템 도출 시 반드시 고려)

이 채널 시청자들은 밈·반어법을 주류 언어로 사용한다.
- "나 망했다 ㅋㅋ", "역시 난 안 돼" 같은 자조적 표현은 공감의 유머이지 부정 반응이 아님
- 댓글의 표면적 의미보다 채널 문화적 맥락을 고려해 해석할 것
- 단, 반어로 포장된 실질적 비판(오류 지적, 주제 불만)은 진지한 피드백으로 처리

### 시청자가 기대하는 것

정확하고 실용적인 투자 정보, 이해하기 쉬운 시각화, 최신 시장 흐름 반영

### 최근 기획 방향성

- 경제의 예능화: '돈'과 관련된 주제를 코미디와 결합하여 진입장벽을 낮춤. 정보 전달보다는 재미와 공감에 방점을 둠.
- 체험형 및 직업 밀착형 콘텐츠: '좋좋슈' 등 특정 직업군이나 경제적 상황을 풍자하거나 체험하는 영상 기획.
- 다양한 전문가 콜라보: 법률, 부동산, 세무 등 실생활 밀착형 분야의 전문가들을 섭외하여 짧고 강렬한 지식 콘텐츠 제작.
- 쇼츠 및 트렌드 최적화: 긴 라이브 편집본 위주인 메인 채널과 달리, MZ세대를 겨냥한 짧고 빠른 호흡의 영상 기획에 집중.

## 지시사항

- 데이터가 존재하는 카테고리에 대해 각 1~2개의 액션 아이템 생성
- 각 아이템은 즉시 실행 가능한 구체적 제안이어야 함
- 채널 특성에 맞는 톤과 방향으로 제안할 것
- JSON 배열로만 응답

### 액션 아이템 유형 (다양하게 활용할 것)

- **다음 영상 주제**: 다룰 만한 후속 주제 또는 시리즈 확장
- **피해야 할 주제/방향**: 시청자 반응을 보아 역효과가 날 수 있는 주제나 접근 방식
- **스토리텔링 방식 조정**: 효과적이었거나 개선이 필요한 전개 구조
- **제목·썸네일 관점**: 클릭을 유도하거나 오해를 줄이기 위한 제목/썸네일 방향
- **신뢰도 관리**: 오류 지적에 대한 후속 대응 또는 정보 검증 강화 방안

### 카테고리 설명

- Topic Value: 주제/내용 만족도 반응 → 시리즈 확장, 전달 방식 조정, 피해야 할 접근
- Expertise & Correction: 전문 지식 보완·오류 지적 → 콘텐츠 깊이 및 신뢰도 조정
- Future Demand: 다음 주제 요청 → 다음 영상 기획 소재 또는 피해야 할 과잉 요청 주제
- Viewer Identity: 시청자 자기 대입·감정 표출 → 썸네일·제목·스토리텔링 방향 설계

        "sourceCategory": "Topic Value"
    }
]
